# Final Deliverables

This folder contains the submission-ready files for the SURV 745 Team Neyman project.

## Contents

- `REPORT_NEYMAN.pdf`: final sampling report
- `REPORT_NEYMAN.tex`: rendered LaTeX file for the final report
- `REPORT_NEYMAN.qmd`: final Quarto source used for the report
- `FRAME1_NEYMAN.pdf`, `FRAME1_NEYMAN.tex`, `FRAME1_NEYMAN.Rmd`, `FRAME1_NEYMAN.csv`: first-stage tract frame deliverables
- `FRAME2_NEYMAN.pdf`, `FRAME2_NEYMAN.tex`, `FRAME2_NEYMAN.Rmd`, `FRAME2_NEYMAN.csv`: second-stage block-group frame deliverables
- `SAMPLE_NEYMAN.pdf`, `SAMPLE_NEYMAN.tex`, `SAMPLE_NEYMAN.Rmd`, `SAMPLE_NEYMAN.csv`: selected-sample deliverables
- `Census Tract FIPS Code Detroit MI_Updated.csv` and `Detroit MI_Updated.xlsx`: updated input data used by the final sources

## Notes

- The repository root still contains earlier draft files kept for reference.
- The files in this folder use the required `NEYMAN` naming so they can be submitted directly.
