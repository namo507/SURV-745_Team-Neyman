\section{Overview}\label{overview}

This file contains the \textbf{selected sample} for the Detroit
Transportation Security Survey. It lists the 40 census tracts selected
as PSUs at Stage 1 and the single block group selected within each tract
as the SSU at Stage 2, along with the census counts, composite measures
of size, inclusion probabilities at each stage, within-block-group
person sampling rates by age domain, and base weights.

We selected the sample using systematic PPS sampling
(\texttt{UPsystematic()} from the R \texttt{sampling} package) with
\texttt{set.seed(-77)}. We sorted tracts in ascending FIPS-code order
before selection. Stage 1 inclusion probabilities were computed using
\texttt{inclusionprobabilities(CompMOS,\ 40)}, which correctly handles
certainty units. Stage 2 block group selection used PPS within each
selected tract.

\section{Notes}\label{notes}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\item
  \textbf{Self-weighting property:} Because
  \(\pi_{g,ij} = \pi_{ij}^{BG} \times f_{g,ij} = f_g\) for all block
  groups \(j\), the base weight \(1/f_g\) is constant within each age
  domain regardless of which tract or block group a respondent falls in.
  The design is self-weighting within each age domain.
\item
  \textbf{Integer allocation:} The operational person counts
  \texttt{n(g)} are derived by a largest-remainder (Hamilton method)
  allocation. Each block group's proportional share of the domain-level
  target (for example, 714 for age 18--34) is floored, and remaining
  units are assigned to block groups with the largest fractional
  remainders. The totals sum exactly to the domain targets.
\item
  \textbf{Base weights and nonresponse adjustment:} The base weights in
  this file are pre-nonresponse-adjustment weights. After data
  collection, a nonresponse adjustment factor should be multiplied into
  the base weight within cells defined by age domain and geographic
  adjustment class. The adjusted weights should then be calibrated to
  2020 Census age-by-sex population totals.
\item
  \textbf{Variance estimation:} For design-based variance estimation,
  the selected census tract (\texttt{Tract\ FIPS}) should be defined as
  the PSU. Analysts should form 20 paired variance-estimation strata by
  pairing adjacent tracts in the PPS-sorted order. The final analysis
  file must include the base weight, PSU identifier, and variance
  stratum variable.
\item
  \textbf{Seed:} \texttt{set.seed(-77)} was used
\end{enumerate}
