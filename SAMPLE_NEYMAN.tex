\section*{Sample Overview}

The Detroit Transportation Security Survey employed a two-stage stratified systematic sampling design with probability proportional to size (PPS) to select 40 primary sampling units (PSUs) represented by Census tracts and 40 secondary sampling units (SSUs) represented by block groups within those tracts. The systematic PPS sampling procedure ensures that larger population units have greater probability of selection, resulting in a self-weighting sample design when implemented correctly. This approach balances operational efficiency with statistical precision in estimating transportation security outcomes across the Detroit metropolitan area.

\section*{Summary Statistics}

\begin{itemize}
\item \textbf{Primary Sampling Units (PSUs):} 40 Census tracts
\item \textbf{Secondary Sampling Units (SSUs):} 40 Block groups
\item \textbf{Sampling Method:} Systematic probability proportional to size (PPS)
\item \textbf{Base Weights by Stratum:}
  \begin{itemize}
    \item Stratum 1: $1/f_g[1] = 225.9$
    \item Stratum 2: $1/f_g[2] = 203.9$
    \item Stratum 3: $1/f_g[3] = 235.5$
    \item Stratum 4: $1/f_g[4] = 228.1$
  \end{itemize}
\item \textbf{Average Workload per PSU:} 53.86 housing units
\end{itemize}

\section*{Notes}

\begin{itemize}
\item \textbf{Tract FIPS:} Federal Information Processing Standards code for Census tract
\item \textbf{Block Group:} Census block group identification code
\item \textbf{Tot Pop:} Total population in the block group
\item \textbf{Adult18+:} Population age 18 years and older
\item \textbf{Age columns:} Count of population in specified age ranges
\item \textbf{$\pi$(Tract):} Probability of selecting the tract in first stage
\item \textbf{$\pi$(BG|Tract):} Conditional probability of selecting the block group given the tract was selected
\item \textbf{$\pi$(Overall):} Overall probability of selection (product of tract and block group probabilities)
\item \textbf{$f$ columns:} Proportion of population in each age group among adults age 18+
\end{itemize}
